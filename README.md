# 📚 Kütüphane Yönetim Sistemi (Console Application)

Bu proje, **C#** programlama dili ve **.NET** platformu kullanılarak geliştirilmiş temel kütüphane operasyonlarını yöneten bir konsol uygulamasıdır. Nesne yönelimli programlama (OOP) prensiplerini ve modern yazılım geliştirme pratiklerini uygulamak amacıyla bir kişisel portfolyo çalışması olarak inşa edilmiştir.

## 🚀 Özellikler
* **CRUD İşlemleri:** Kitap ekleme ve listeleme fonksiyonları.
* **Veri Kalıcılığı (Persistence):** Veriler `kitaplar.txt` dosyasında saklanır; program kapatılsa da veriler kaybolmaz.
* **Modern C# Standartları:** `required` property, `null-coalescing` ve `Object Initializer` kullanımı.
* **Hata Yönetimi:** `int.TryParse` ile kullanıcı girişlerinin güvenli kontrolü.

## 🛠 Kullanılan Teknolojiler
* **Dil:** C# (.NET 8.0+)
* **Dosya Sistemi:** System.IO (Text-based storage)

## 📖 Nasıl Çalıştırılır?
1. Projeyi bilgisayarınıza indirin veya clone'layın.
2. Visual Studio veya VS Code ile açın.
3. `F5` tuşuna basarak projeyi derleyin ve çalıştırın.
